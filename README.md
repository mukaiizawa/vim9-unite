# vim9-unite

This project is inspired by [unite.vim](https://github.com/Shougo/unite.vim).

However, it differs in a few major ways:

- It is implemented in Vim9 script.
- It does not depend on a specific filer such as [vimfiler.vim](https://github.com/Shougo/vimfiler.vim).
- It is not designed for deep customization, but aims to be reasonably usable after installation.
